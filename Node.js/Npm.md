# npm(Node Package Manager)

- Node.js 환경에서 JavaScript 런타임을 위한 기본 패키지 매니저.

## package.json

- `npm install` 명령어를 입력했을 때 생성되는 패키지 버전 명시 파일.

## package-lock.json

- `npm`이 `node_modules` 또는 `package.json`을 수정하는 작업들이 자동으로 저장되는 파일.
