# Type System

- 프로그램을 실행하기 전에 표시 오류가 발생하지 않을 것인지 자동으로 결정한다.

- 정확성: 논리적 오류(컴파일 에러, 런타임 오류 등)를 감지한다.
- 성능: 컴파일러 최적화를 허용한다.
- 문서화: 변수, 매개변수 및 반환 값의 의미론적 의미 전달 → 개발자에게 추상화를 제공한다.
