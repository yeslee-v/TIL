# change image size in markdown

- 기본 이미지 삽입

  ```markdown
  ![title](pic.jpg)
  ```

- markdown 내에서 몇몇 html 문법을 사용할 수 있다.

  ```markdown
  <img src="pic.jpg" alt="title" width="100" height="100"/>
  ```

  - 가능하지만, GitHub에서 지원하지 않는 형식이다.

  ```markdown
  <img src="pic.jpg" alt="title" style="width:100px;height=100px"/>
  ```
